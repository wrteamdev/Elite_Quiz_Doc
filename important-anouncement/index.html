Important Announcement: Changes to Premium Categories

We're making significant changes to how Premium Categories and Subcategories work starting with version v2.2.3. To give you time to prepare, we're informing you now so you can decide whether to update to the new version or stay on the current one. What's changing? Currently, you can make both Categories and Subcategories premium. While this offers flexibility, it's caused complexity issues in the app and made it difficult to manage different quiz features like Battles and Self Challenges. To simplify the app and improve performance, we're changing the system so that only Categories can be premium. Subcategories will no longer have premium options. What does this mean for you?
  • - If you want to continue using the current system with premium subcategories, you can stay on version v2.2.2.
  • - If you're okay with only premium Categories, you can update to version v2.2.3 when it's released very soon.
    • We understand this change might affect your quiz app, so you have time to adjust your Categories and Subcategories before the update. To help you visualise the differences, we've included diagrams showing how the system works now and how it will work after the change.
